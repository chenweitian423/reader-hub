私有站点接入

适合接入你自己维护或明确授权的小说站点。先测试搜索，再一键导入成书源。

常见类型
高级规则设置

规则可写 JSON 路径、CSS 选择器或 XPath 表达式。搜索地址里用 {keyword} 作为关键词占位符。

CSS / HTML 站 .book-list .item .title@text / a@href / #content@html
XPath 站 //div[@class='book-item'] .//a/text() / .//a/@href / //div[@id='content']
JSON API 站 data.list / data.book / data.chapters title / author / content
常见属性写法 @text 取文本，@html 取 HTML，@href / @src 取链接与图片 目录翻页常见：.next@href，正文翻页常见：.page-next@href
这里会显示测试命中的书籍预览，以及当前站点是否支持目录和正文抓取。
