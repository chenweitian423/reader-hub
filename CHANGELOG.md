# Changelog

## 1.7.0 - 2026-04-23

- 首页继续朝正式读书产品演进，新增“焦点书籍”和“继续阅读”模块。
- 首页现在会根据当前阅读、搜索结果或书架内容，自动展示更适合打开的一本书。
- 新增最近在读卡片区，支持从首页直接续读书架里的书。
- 优化首页视觉层次，让搜书页更像书城首页而不是管理后台。

## 1.6.0 - 2026-04-23

- 重做前端信息架构：首页现在只保留搜书、搜索结果和阅读器。
- 新增左侧菜单导航，把书架、书源管理、数据工具拆成独立页面，不再挤在同一屏。
- 优化整体视觉样式，增强菜单层级、首页聚焦感和移动端布局表现。
- 首页新增快捷搜索词按钮，书架继续阅读会自动切回搜书首页。

## 1.5.1 - 2026-04-23

- 修复 Docker 镜像未包含 `VERSION` 文件导致服务接口返回版本号 `0.0.0` 的问题。
- 发布镜像和服务器部署现在会正确显示应用版本号。

## 1.5.0 - 2026-04-23

- 新增应用版本文件 `VERSION`，后端、文档和 Docker 发布流程统一读取版本号。
- 新增 `CHANGELOG.md`，每次迭代都可以追加版本更新说明。
- 新增数据备份与恢复能力，支持导出书源、书架、缓存章节和阅读设置，并支持合并导入或覆盖恢复。
- Docker Hub 镜像发布新增版本标签，除了 `latest` 和 `sha-*` 以外，还会生成 `1.5.0` 与 `1.5` 这样的版本 tag。
- `docker-compose.yml` 支持通过 `READER_HUB_IMAGE_TAG` 固定部署指定镜像版本。
