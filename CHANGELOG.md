# Changelog

## 1.21.0 - 2026-04-24

- 前台和后台界面正式拆开：普通用户现在拥有更像小说站的“书城首页 / 我的书架 / 个人中心”结构，管理员继续保留书源、用户和数据工具页面。
- 普通用户已开放书架、阅读进度、本地导入、缓存和阅读设置等核心读书能力，能完整走通“登录 -> 搜书 -> 加书架 -> 继续阅读”的链路。
- 首页文案、侧边导航和个人中心重做为更接近小说站前台的风格，突出搜索、最近在读、更新分区和账号阅读习惯，而不再像后台工具页。

## 1.20.0 - 2026-04-24

- 新增轻量账号体系，内置管理员与普通用户两种角色。管理员可进入管理界面维护书源、用户和数据工具；普通用户只保留搜书与阅读界面。
- 新增登录界面、当前账号状态和退出登录能力，并增加管理员专属的“用户管理”页面，可创建用户、切换角色、停用账号和重置密码。
- 默认种子账号已写入启动流程：管理员 `admin / admin123`，普通用户 `reader / reader123`，并可通过环境变量覆盖。

## 1.19.9 - 2026-04-24

- 优化独立阅读页的“目录抽屉”显示区域，抽屉更宽、更贴边，列表滚动区和头部留白更均衡。
- 章节项改成更高的可点击卡片，长标题在目录抽屉里显示更完整，不再出现文字被裁切、贴边或显示不全的问题。
- 当前激活章节在目录抽屉里补充了更清晰的高亮阴影，移动端查看时更容易定位当前位置。

## 1.19.8 - 2026-04-24

- 修复 `1.19.7` 中阅读器顶部改版后前端脚本启动即报错的问题。顶部旧的翻页按钮已从界面移除，但脚本里仍然直接绑定事件，导致页面停在 `v0.0.0` 且所有交互失效。
- 阅读器翻页状态现在会兼容“仅保留底部翻页按钮”的布局，顶部按钮不存在时不会再触发空指针错误。

## 1.19.7 - 2026-04-24

- 独立阅读页顶部进一步收成正式阅读器样式，现在只保留“返回 + 书名 + 当前章 + 更多”四块核心信息，其他操作统一收进右上角更多菜单。
- 顶部标题栏在滚动后会继续保留当前章节信息，并收成更轻的透明形态，避免一整块说明区域持续占用阅读空间。
- 阅读页正文工具头同步简化为“当前章节 + 进度”，和新的紧凑顶部栏配合使用时更像成熟阅读器。

## 1.19.6 - 2026-04-24

- 阅读页顶部说明卡改成“滚动后超窄透明标题栏”，下滑时不再整块压住正文，只保留紧凑书名与操作按钮。
- 阅读页正文功能栏改为静态布局，不再跟随滚动，解决正文区域被工具栏盖住一部分的问题。
- 左侧章节目录新增折叠/展开控制，并补了章节标题换行与宽度限制，长章节名不再把侧栏撑穿。

## 1.19.5 - 2026-04-24

- 调整独立阅读页顶部布局，阅读页说明卡会固定在页面最上方并收紧尺寸，减少滚动时对正文的遮挡感。
- 正文工具条下移，为固定顶部栏留出更明确的安全间距，避免阅读时标题区和工具条互相压住。
- 阅读页右下角新增“回到顶部”悬浮按钮，滚动一定距离后自动出现，可一键快速返回顶部。

## 1.19.4 - 2026-04-24

- 把“笔趣阁前端站”继续细分成两个模板：`标准版` 和 `严格请求头版`。
- 标准版适合大多数前端页面 + JSON API 的笔趣阁类站点；严格请求头版额外带上 `Accept-Language`、`Origin`、缓存控制等头，适合对请求特征更敏感的同类站点。
- “填入示例”现在默认会加载 `笔趣阁前端站（标准版）`，更适合作为同类站点接入起点。

## 1.19.3 - 2026-04-24

- 私有站点接入新增“笔趣阁前端站”专用模板，适合这类前端页面 + JSON API 的小说站点一键套用。
- “填入示例”现在会优先填充笔趣阁前端站模板，直接带上常用请求头、搜索规则、目录规则和正文规则。
- 这样同类站点接入时，可以先点模板再微调域名与关键词，不必每次都依赖自动识别。

## 1.19.2 - 2026-04-24

- 修复一类“前端页面 + JSON API”小说站的自动识别失败问题，像 `bqg496.xyz` 这类站点现在会直接识别成 `api/search + api/book + api/booklist + api/chapter` 链路，而不再误判成普通 HTML 搜索页。
- 私有站点导入器现在能把这类站点自动转换成原生请求链路，测试搜索、导入书源、打开目录和读取正文会走统一的 JSON API 配置。
- 通用映射执行器新增对“字符串列表”结果的兼容，目录接口只返回章节标题数组时，也能自动生成章节序号并继续阅读。

## 1.19.1 - 2026-04-24

- 修复一类站点“自动识别后测试搜索直接 404”的问题。
- 自动识别现在会优先探测页面里的搜索表单、搜索链接和常见搜索路径，尽量回填真实可访问的搜索入口，而不再盲猜固定的 `/search`。

## 1.19.0 - 2026-04-24

- 私有站点接入器新增“小说网址自动识别”入口，粘贴书籍详情页或章节页网址后，系统会自动猜测站点类型并尽量回填规则。
- 自动识别现在会尝试推断站点首页、搜索表单、详情页元数据、目录列表、目录翻页、正文区域和正文翻页规则。
- 自动识别结果会直接回填到表单，并在预览区展示本次猜测的提示信息，方便继续测试和微调。

## 1.18.0 - 2026-04-24

- 私有站点接入器新增可视化高级设置，详情页元数据规则、目录下一页规则和正文下一页规则现在都可以直接在表单里配置。
- 新增五类常见站点模板，覆盖 PC HTML 列表站、移动站多页正文、JSON API 站、JSON 搜索 + HTML 阅读，以及 XPath 站点。
- 新增规则速查卡片和更丰富的测试预览标签，能直接看出当前站点是否已配置详情元数据、目录翻页和正文翻页能力。

## 1.17.0 - 2026-04-24

- 书源管理新增“私有站点接入”卡片，支持通过表单填写搜索、目录和正文规则，先测试再导入书源。
- 后端新增私有站点预览、测试和导入接口，适合接入你自己维护或明确授权的小说站点。
- 站点测试结果现在会直接在页面里展示命中书籍预览，并标记当前规则是否已经具备阅读能力。

## 1.16.0 - 2026-04-24

- 书源管理新增多选删除能力，支持本页全选、清空选择和删除所选书源。
- 新增“删除全部”动作，可一键清空全部可删除书源，系统内置的“本地导入书库”会自动保留。
- 书源列表卡片新增勾选列与批量操作区，删除类操作的禁用态和提示文案也一并补齐。

## 1.15.1 - 2026-04-24

- 修复前端静态资源缓存导致的“新页面结构配旧脚本”问题。
- 首页现在会为 `favicon`、`styles.css`、`app.js` 自动追加版本号查询参数，版本更新后浏览器会更可靠地拉取新资源。
- 连带修复书源管理分页控件显示了但仍按旧逻辑渲染全部书源的问题。

## 1.15.0 - 2026-04-24

- 书架页新增两块独立入口：本机导入区保留拖拽与目录选择，局域网设备入口单独展示上传页链接。
- 本地导入书和局域网上传书现在会保存不同的导入来源标记，并在书架与详情里显示。

## 1.14.2 - 2026-04-24

- 调整 `/api/library/uploads` 的浏览器访问行为：现在直接在浏览器打开这个地址，会返回一个可直接选择文件、选择目录、拖拽上传的网页，而不再只显示 JSON 接口说明。
- 保留原有接口能力，非浏览器或非 HTML 请求仍然可以继续拿到 JSON 说明和 POST 上传接口。

## 1.14.1 - 2026-04-24

- 新增全页拖拽防误触处理，拖文件到页面其他位置时不再容易把当前页面直接替换成文件内容。
- 书源管理新增分页和每页数量控制，大量书源时不再一屏拉得过长；书源说明默认压成两行，页面更紧凑。

## 1.14.0 - 2026-04-24

- 重做书架页本地导入区域，新增独立投放面板，支持点击选择文件、点击选择整个目录，以及直接拖动文件或文件夹上传。
- 新增待上传文件队列与文件清单展示，上传前可以先确认来源目录、文件数量和体积，再一键导入到书架。
- 拖动上传现在会自动递归读取目录内容，并过滤不支持的格式，只保留 TXT、MD、EPUB。

## 1.13.1 - 2026-04-24

- 新增跨域支持，方便同一网络下的其他网页、小工具或表单直接调用上传接口。

## 1.13.0 - 2026-04-23
